4. Troubleshooting Steps

If webhook is not working:

  1. Check if WEBHOOK_URL environment variable is set in Vercel
  2. Verify your webhook URL is accessible and accepts POST requests
  3. Check Vercel function logs for error messages
  4. Test your webhook URL manually with a tool like Postman
  5. Make sure your webhook endpoint returns a 200 status code
